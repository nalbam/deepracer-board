# deepracer-board

## Install and configure the Amplify CLI.

```bash
npm install -g @aws-amplify/cli
```

## amplify

```bash
amplify init
amplify status

amplify auth add
amplify analytics add
amplify api add
amplify push

amplify hosting add
amplify publish
```
